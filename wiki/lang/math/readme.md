# Mathematical Operations

Math operations work exactly as you expect them to, with both symbols and words being usable.

	+
		plus  
		added to  
		add <$var> to
		and
	
	-  
		minus  
		subtracted from
		subtract <$var> by
	
	/  
		divided by  
		divide <$var> by 
	
	*  
		multiplied by  
		multiply <$var> by
	
	++  
		increment <$var> (default 1)
		increment <$var> by 
	
	--  
		decrement <$var> (default 1)
		decrement <$var> by
